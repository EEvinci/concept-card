# 知识概念卡片生成器 (Concept Card Generator)

**通用知识卡片 Skill** — 输入任意主题，一句话生成专业深色风格视觉卡片（1200×675px）。

支持：**投资、金融、心理学、历史、科学、哲学、编程、语言**等任何主题。

## 效果预览

> 纯黑金配色：深蓝黑底 (#0a0e1a) + 金色点缀 (#f0b429)

## 快速使用

```
"帮我做一张关于复利的卡"
"生成一张锚定效应的概念卡"
"做一张关于量子力学的科普卡"
"帮我做一张海龟汤的规则介绍卡"
```

## 配色规范

| 元素 | 颜色 | 用途 |
|------|------|------|
| 背景 | `#0a0e1a` | 深蓝黑底 |
| 金色 | `#f0b429` | 标题/公式/关键词 |
| 白色 | `#ffffff` | 大标题 |
| 灰色 | `#9ca3af` | 次要文字 |
| 绿色 | `#22c55e` | 结论/正确选项 |
| 红色 | `#f87171` | 误区/警示 |

## 卡片结构

```
┌─────────────────┬─────────────────┬─────────────────┐
│  左侧：定义区   │  中间：3要点    │  右侧：案例    │
│  标题 + 公式    │  思维模型      │  具体算例      │
│  误区 + 金句    │  或关键机制     │  对比 + 结论   │
└─────────────────┴─────────────────┴─────────────────┘
```

## 支持的主题类型

| 类型 | 示例 |
|------|------|
| 投资/金融 | 复利、机会成本、安全边际、夏普比率 |
| 心理学 | 锚定效应、损失厌恶、认知偏差 |
| 科学 | 量子隧穿、光的折射、热力学 |
| 历史 | 明治维新、文艺复兴 |
| 哲学 | 奥卡姆剃刀、帕斯卡赌注 |
| 编程 | 递归、快排、API设计 |
| 生活 | 海龟汤、剧本杀、三门问题 |

## 设计原则

- ❌ 无 Day 序号标注
- ❌ 无个人品牌标识
- ✅ 纯黑金配色，无杂色干扰
- ✅ 三栏逻辑流：定义 → 机制 → 案例
