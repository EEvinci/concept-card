# 内容类型结构参考

## 1. 投资 / 金融

```
左：主标题 + 公式 + 一句话定义 + 误区 + 金句
中：隐形成本 / 预期收益 / 沉没成本谬误
右：算例（10万现金/两个投资方案对比）
标签：#投资思维 #机会成本 #决策框架
金句："The cost of any choice is what you sacrifice."
```

## 2. 心理学 / 认知偏差

```
左：主标题 + 一句话定义 + 误区 + 金句
中：三个认知效应（如锚定效应 / 损失厌恶 / 确认偏误）
右：生活中的实际案例（购物/投资/决策场景）
标签：#行为金融 #认知偏差 #心理模型
金句："We don't see things as they are, we see them as we are."
```

## 3. 科学 / 技术原理

```
左：主标题 + 核心原理（公式）+ 一句话 + 误区
中：原理A / 原理B / 原理C（三个关键机制）
右：生活应用案例（手机充电/汽车发动机/天气预报）
标签：#科学原理 #技术科普 #为什么
金句："Everything should be made as simple as possible, but not simpler."
```

## 4. 历史 / 事件

```
左：事件名称 + 背景 + 误区（常见误读）+ 金句
中：原因 / 经过 / 影响（三个维度）
右：时间线（年表形式，用.timeline-item）
标签：#历史 #事件 #教训
```

## 5. 哲学 / 思想

```
左：核心问题 + 流派对比 + 结论 + 金句
中：三个主要学派/观点（如存在主义/功利主义/康德哲学）
右：生活中的实践（如何在日常做决策）
标签：#哲学 #思维 #人生
金句："The unexamined life is not worth living." — Socrates
```

## 6. 编程 / 技术

```
左：概念名称 + 核心原理 + 一句话 + 误区（"初学者常犯"）
中：三个要点（使用场景 / 优缺点 / 替代方案）
右：代码示例（伪代码或关键片段）
标签：#编程 #技术 #工程
金句："Programs must be written for people to read." — Harold Abelson
```

## 7. 通用深度（默认）

```
左：主标题 + 核心定义 + 误区 + 金句
中：是什么 / 为什么重要 / 实际应用
右：案例解析（具体场景）
标签：#学习 #认知 #概念
```
